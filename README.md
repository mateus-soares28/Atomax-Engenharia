# Atomax - Engenharia Elétrica

Site institucional da Atomax, empresa especializada em engenharia elétrica, laudos técnicos e locação de equipamentos.

## 📋 Páginas

- **index.html** - Página inicial com informações da empresa
- **equipamentos.html** - Catálogo de equipamentos com carrinho de solicitação
- **laudos.html** - Seção de laudos e certificações

## 🛠️ Tecnologias

- HTML5
- CSS3 (com design responsivo)
- JavaScript vanilla
- Font Awesome (ícones)
- Google Fonts (Poppins)

## 📁 Estrutura

```
css/
  ├── reset.css
  ├── responsive.css
  └── style.css

js/
  ├── script.js
  ├── menu.js
  ├── catalogo.js
  └── laudo.js

assets/
  └── img/
```

## 🚀 Como usar

1. Abra `index.html` no navegador
2. Navegue pelas páginas pelo menu
3. Explore o catálogo de equipamentos
4. Use o carrinho para solicitar itens

## ⚙️ Funcionalidades

- Menu responsivo
- Catálogo de equipamentos interativo
- Carrinho de solicitações
- Layout adaptável para mobile e desktop
